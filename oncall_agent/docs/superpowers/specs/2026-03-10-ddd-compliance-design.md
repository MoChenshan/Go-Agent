# DDD Compliance Refactor ‚Äî Design Spec

**Date:** 2026-03-10
**Status:** Approved

---

## Problem

The current `oncall_agent` project violates the `trpc-ddd-codegen` skill requirements in 5 areas:

1. **Domain imports infrastructure** ‚Äî `domain/tools/*` import `infrastructure/external/http/*` and `infrastructure/repo/mysql/magic_config`, leaking infra types into domain interface signatures.
2. **No `domain/interfaces/`** ‚Äî Shared external service interfaces (`GalileoAPI`, `LingshanAPI`, `CdkeyAPI`, `ConditionLogAPI`, `MagicCliAPI`, `MagicConfigAPI`) are defined inside individual domain tool packages, causing duplication (`GalileoAPI` appears in both `log_query` and `trace_analysis`).
3. **Infra types in domain model** ‚Äî Request/response types for galileo, lingshan, cdkey, and magic_config DB entities live in infra packages, not `domain/model`.
4. **No Wire DI** ‚Äî `main.go` (398 lines) manually wires all dependencies via `initServers()`.
5. **No README troubleshooting guide** ‚Äî Required by the skill checklist.

## Interface Definitions

All interfaces in `domain/interfaces/external/` use only `domain/model` types:

```go
// galileo_api.go
type GalileoAPI interface {
    QueryLog(ctx context.Context, req *model.QueryLogReq) (*model.QueryLogRsp, error)
    QueryTrace(ctx context.Context, req *model.QueryTraceReq) (*model.QueryTraceRsp, error)
}

// lingshan_api.go
type LingshanAPI interface {
    GetSrvDetailByNames(ctx context.Context, req model.GetSrvDetailByNamesReq) (model.GetSrvDetailByNamesRsp, error)
}

// cdkey_api.go
type CdkeyAPI interface {
    BatchQueryCdkey(ctx context.Context, req model.BatchQueryCdkeyReq) (model.BatchQueryCdkeyRsp, error)
}

// conditionlog_api.go
type ConditionLogAPI interface {
    GetConditionLog(ctx context.Context, start, end int64, traceID string) (string, error)
}

// magiccli_api.go
type MagicCliAPI interface {
    EncryptMagicID(ctx context.Context, plain string) (string, error)
}

// magic_config_api.go ‚Äî all methods use model.ActInfo, model.ModuleInfo, etc.
type MagicConfigAPI interface {
    GetActInfoByID(ctx context.Context, id int) (*model.ActInfo, error)
    // ... (full set of existing methods)
}
```

## Wire Dependency Injection

### `wire.go` pattern

```go
//go:build wireinject

package main

func InitApp(cfg rainbow.AppConfig) (*App, error) {
    wire.Build(
        // infrastructure providers
        provideGalileoCli,      // wraps galileocli.New(cfg.BkAppCode, cfg.BkAppToken)
        provideLingshanCli,     // wraps lingshancli.New(cfg.XGatewaySecretID, cfg.XGatewaySecretKey)
        provideCdkeyCli,        // wraps cdkeycli.New(cfg.ESUsername, cfg.ESPassword, cfg.FlowPath)
        conditionlog.New,
        magiccli.New,
        provideMagicConfigDB,   // wraps gorm.NewClientProxy + magicconfig.New
        provideWujiCli,         // wraps initWuji()

        // domain tools
        provideMCPTool,
        provideTraceAnalysisDep,   traceanalysis.New,
        provideLogQueryDep,        logquery.New,
        provideLingshanQueryDep,   lingshanquery.New,
        provideCdkeyQueryDep,      cdkeyquery.New,
        provideMagicToolDep,
        // util tools (base64, timestamp) provided as named values

        // domain agents
        provideSpanAnalysisDep,    spananalysis.NewSpanAnalysisAgentTool,
        provideRepoAgentDep,       repoagent.NewRepoAgentTool,
        provideMagicOncallDep,     magicagent.New,
        provideRuleEngineDep,      ruleengine.New,
        provideCdkeyAgentDep,      cdkagent.New,
        provideMagicConfigAgentDep, magicconfigagent.New,

        // session
        provideSessionService,

        // services
        provideSSEServers,
        provideA2AServers,
        provideAguiServers,
        debug.New,

        wire.Struct(new(App), "*"),
    )
    return nil, nil
}
```

### `LocalTools []tool.Tool` per agent

Each agent's `LocalTools` varies, so Wire cannot auto-wire it. Each agent gets a `provide*Dep` helper:

```go
func provideMagicOncallDep(
    m *openai.Model, w domainmodel.WujiAPI, mcp mcptool.API, gc domainmodel.GenConfig,
    trace, logq, span, repo, b64, lingshan, dt2ts, ts2dt, modType, actInfo, proposeChange tool.Tool,
) magicagent.Dep {
    return magicagent.Dep{
        ModelInstance: m, WujiCli: w, MCPTool: mcp, GenConfig: gc,
        LocalTools: []tool.Tool{trace, logq, span, repo, b64, lingshan, dt2ts, ts2dt, modType, actInfo, proposeChange},
    }
}
```
