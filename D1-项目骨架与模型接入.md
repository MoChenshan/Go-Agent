## 一、环境准备

### 1.1 Go 环境安装

```bash
# 确认 Go 版本 >= 1.21（框架要求 >= 1.19，推荐 1.21+）
go version
# 期望输出：go version go1.21.x windows/amd64

# 配置 Go 代理（内网加速）
go env -w GOPROXY=https://goproxy.cn,direct
go env -w GOPRIVATE=git.woa.com,trpc.group

# 确认 GOPATH
go env GOPATH
# 记住这个路径，后面 CLI 工具会安装到 $GOPATH/bin
```

**验收**：`go version` 输出 `go1.21+`

### 1.2 安装 tRPC 命令行工具

```bash
# 安装 trpc CLI（用于生成项目骨架）
go install trpc.tech/trpc-go/trpc-go-cmdline/v2/trpc@latest

# 验证安装
trpc version

# 如果提示 trpc 不是可识别的命令，需要将 GOPATH/bin 加到 PATH：
# Windows PowerShell:
$env:PATH += ";$(go env GOPATH)\bin"
# 或永久添加到系统环境变量
```

**验收**：`trpc version` 正常输出版本号

### 1.3 配置 LLM API 密钥

```bash
# === 方式一：混元（内网优先） ===
# 在 PowerShell 中设置环境变量（当次会话生效）
$env:OPENAI_API_KEY = "your-hunyuan-api-key"
$env:OPENAI_BASE_URL = "http://hunyuanapi.woa.com/openapi/v1"

# === 方式二：OpenAI / DeepSeek（外网备选） ===
$env:OPENAI_API_KEY = "sk-xxxxxxxxxxxxxxxx"
$env:OPENAI_BASE_URL = "https://api.openai.com/v1"

# === 方式三：太极平台 ===
$env:OPENAI_API_KEY = "your-taiji-api-key"
$env:OPENAI_BASE_URL = "http://api.taiji.woa.com/openapi"
```

> ⚠️ **重要**：tRPC-Agent-Go 的 OpenAI 适配器会自动从 `OPENAI_API_KEY` 和 `OPENAI_BASE_URL` 环境变量读取配置。

**验收**：`echo $env:OPENAI_API_KEY` 和 `echo $env:OPENAI_BASE_URL` 有值

## 三、编写 trpc_go.yaml 基础配置

创建 `trpc_go.yaml`，D1 只配单端口 8080：

```yaml
# trpc_go.yaml - GameOps Agent 服务配置（D1 基础版）
global:
  namespace: Development
  env_name: test

server:
  app: gameops
  server: agent
  service:
    - name: trpc.gameops.agent.AGUI
      network: tcp
      protocol: http
      ip: 0.0.0.0
      port: 8080
      timeout: 60000    # 60s，Agent 调用链较长需要足够超时

client:
  service:
    - name: llm_client
      protocol: http
      timeout: 0        # 流式输出不限超时
      conn_type: httppool
      httppool:
        idle_conn_timeout: 50s
        max_idle_conns_per_host: 100
```

### 4.1 Coordinator Agent 基础 System Prompt

创建文件 `agent/prompt/coordinator.txt`：

```
你是 GameOps 智能运维助手，专门帮助运维工程师和开发人员排查游戏服务器问题。

2. 提供故障排查建议

请用中文回复，保持专业、简洁。
如果用户的问题超出你的能力范围，请诚实告知。
```

### 4.2 Coordinator Agent 代码

创建文件 `agent/coordinator.go`：

```go
package agent

import (
	"os"

	"trpc.group/trpc-go/trpc-agent-go/agent/llmagent"
	"trpc.group/trpc-go/trpc-agent-go/model/openai"
)

// NewCoordinator 创建协调者 Agent（D1 基础版：仅做简单对话，不挂子 Agent）
func NewCoordinator() *llmagent.Agent {
	// 读取 System Prompt
	prompt, err := os.ReadFile("agent/prompt/coordinator.txt")
	if err != nil {
		prompt = []byte("你是 GameOps 智能运维助手，请用中文回复。")
	}

	// 创建 OpenAI 兼容模型（自动读取 OPENAI_API_KEY 和 OPENAI_BASE_URL 环境变量）
	model := openai.New("hunyuan-a13b") // 或 "gpt-4o" / "deepseek-chat"

	// 创建 LLMAgent
	coordinator := llmagent.New("coordinator",
		llmagent.WithModel(model),
		llmagent.WithInstruction(string(prompt)),
	)

	return coordinator
}
```

> **模型选择说明**：
> - 混元：`hunyuan-a13b`（内网，需配 `OPENAI_BASE_URL=http://hunyuanapi.woa.com/openapi/v1`）
> - OpenAI：`gpt-4o`
> - DeepSeek：`deepseek-chat`
> - 太极平台：可用 `taiji.NewOpenAI()` 替代 `openai.New()`
>
> 根据你实际可用的模型修改此处。

### 4.3 主入口 main.go

创建文件 `main.go`：

```go
package main

import (
	"context"
	"fmt"
	"os"
	"os/signal"
	"syscall"

	"gameops-agent/agent"

	"trpc.group/trpc-go/trpc-agent-go/runner"
	"trpc.group/trpc-go/trpc-agent-go/server/agui"
	trpc "trpc.group/trpc-go/trpc-go"
)

func main() {
	// 1. 创建 Coordinator Agent
	coordinator := agent.NewCoordinator()

	// 2. 创建 Runner（执行引擎）
	r := runner.NewRunner("gameops-agent", coordinator)

	// 3. 注册 AG-UI 服务（自带 Web 前端）
	agui.RegisterAgent(coordinator, r,
		agui.WithTitle("GameOps 智能运维助手"),
		agui.WithDescription("LetsGo 游戏服务器智能运维问答系统"),
	)

	// 4. 启动 tRPC 服务
	s := trpc.NewServer()

	fmt.Println("========================================")
	fmt.Println("  GameOps Agent 启动成功！")
	fmt.Println("  AG-UI 前端: http://localhost:8080")
	fmt.Println("========================================")

	// 优雅退出
	ctx, cancel := context.WithCancel(context.Background())
	defer cancel()

	go func() {
		sigCh := make(chan os.Signal, 1)
		signal.Notify(sigCh, syscall.SIGINT, syscall.SIGTERM)
		<-sigCh
		fmt.Println("\n正在关闭服务...")
		cancel()
	}()

	if err := s.Serve(); err != nil {
		fmt.Printf("服务启动失败: %v\n", err)
		os.Exit(1)
	}

	<-ctx.Done()
}
```

> ⚠️ **注意**：以上代码基于框架 wiki 文档的 API 接口编写。实际 import path 可能因框架版本略有不同，`go mod tidy` 时如果报错，参考框架 `examples/llmagent/main.go` 示例调整。

**方式三：参考框架 examples/llmagent 的交互式终端方式**

如果 AG-UI 注册方式与你的框架版本不兼容，可以先退化为终端交互模式验证模型连通性：

```go
// main_cli.go（临时验证文件，确认模型可用后删除）
package main

import (
	"bufio"
	"context"
	"fmt"
	"os"

	"trpc.group/trpc-go/trpc-agent-go/agent/llmagent"
	"trpc.group/trpc-go/trpc-agent-go/model/openai"
	"trpc.group/trpc-go/trpc-agent-go/runner"
)

func main() {
	model := openai.New("hunyuan-a13b")
	agent := llmagent.New("coordinator",
		llmagent.WithModel(model),
		llmagent.WithInstruction("你是GameOps智能运维助手，请用中文回复。"),
	)
	r := runner.NewRunner("gameops-agent", agent)

	scanner := bufio.NewScanner(os.Stdin)
	fmt.Println("GameOps Agent 已启动（输入 exit 退出）")

	for {
		fmt.Print("\n👤 你: ")
		if !scanner.Scan() {
			break
		}
		input := scanner.Text()
		if input == "exit" {
			break
		}

		ctx := context.Background()
		eventCh, err := r.Run(ctx, "user1", "session1", input)
		if err != nil {
			fmt.Printf("错误: %v\n", err)
			continue
		}

		fmt.Print("🤖 Agent: ")
		for evt := range eventCh {
			if evt.Content != "" {
				fmt.Print(evt.Content)
			}
		}
		fmt.Println()
	}
}
```
