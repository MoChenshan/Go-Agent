## 概述

你可以运行已经实现好的 Python 数学计算脚本，快捷完成用户的计算需求，结果输出到文件以便框架收集。

## 命令示例

### 1) 计算斐波那契数列

```bash
python3 scripts/fib.py 10 > out/fib.txt
```

### 2) 计算素数

```bash
python3 scripts/primes.py 100 > out/primes.txt
```

## 输出文件

- out/fib.txt - 斐波那契数列结果
- out/primes.txt - 素数列表
